# گیاه‌یار 🌿

اپلیکیشن اندروید برای ثبت و پیگیری کارهای مربوط به گیاهان و گلدون‌ها: آبیاری، کوددهی، هرس، تعویض گلدان و... با تاریخ شمسی.

## امکانات
- ثبت هر گیاه با نام، نوع، عکس، دوره آبیاری و کوددهی
- ثبت تاریخچه‌ی کامل کارها با تاریخ شمسی
- یادآوری خودکار (نوتیفیکیشن) وقتی زمان آبیاری/کوددهی فرا برسد
- گرفتن بکاپ کامل (اطلاعات + عکس‌ها) به‌صورت فایل zip و بازیابی آن
- بدون نیاز به اینترنت، تمام اطلاعات روی خود گوشی ذخیره می‌شود

## اجرا در Android Studio
1. پوشه پروژه را در Android Studio باز کنید (Open).
2. Android Studio به‌طور خودکار Gradle Wrapper لازم را می‌سازد (یا از منوی
   `File > Sync Project with Gradle Files` استفاده کنید).
3. دکمه Run را بزنید.

> این پروژه عمداً بدون فایل باینری `gradle-wrapper.jar` ارائه شده؛
> هم Android Studio و هم پایپ‌لاین GitHub Actions به‌صورت خودکار Gradle مناسب را
> فراهم می‌کنند، بدون نیاز به فایل باینری در ریپازیتوری.

## ساخت خودکار APK با GitHub Actions
با هر `push` به شاخه‌ی `main`، فایل `.github/workflows/build-apk.yml`:
1. یک APK دیباگ می‌سازد و به‌عنوان Artifact قابل دانلود در تب Actions قرار می‌دهد.
2. یک APK ریلیز (بدون امضا) نیز می‌سازد.

برای دانلود: به تب **Actions** ریپازیتوری بروید → آخرین اجرای موفق → بخش
**Artifacts** پایین صفحه → دانلود `plantcare-debug-apk`.

### اگر خواستید APK ریلیز امضاشده و آماده انتشار بسازید
باید یک keystore بسازید و اطلاعات آن را به‌صورت GitHub Secrets اضافه کنید،
سپس مرحله‌ی امضا را به workflow اضافه کنم — کافیست بگویید تا این بخش را هم برایتان تکمیل کنم.

## ساختار پروژه
```
app/src/main/java/ir/plantcare/app/
  data/        # Entity ها و Room DAO/Database
  util/        # تبدیل تاریخ شمسی، بکاپ، عکس، نوتیفیکیشن
  ui/          # صفحات Jetpack Compose
```

## نکات فنی
- تقویم شمسی بدون هیچ وابستگی بیرونی و با الگوریتم تبدیل مستقیم پیاده شده است.
- عکس‌ها در حافظه‌ی داخلی خود اپ (`filesDir/photos`) نگه‌داری می‌شوند تا بکاپ/بازیابی
  ساده و قابل‌اعتماد باشد.
- حداقل نسخه اندروید پشتیبانی‌شده: Android 7.0 (API 24).
